# _CalculadoraSimple_

**_Este es un proyecto en donde se crea un menú principal para poder visualizar las funcionalidades de una calculadora, más bien la de sumas, restas, multiplicaciones y divisiones entre varios números en donde sea necesario._**

## **_Con esta calculadora básica consta de 4 operaciones muy sencillas de realizar..._**

- **_Suma (Adición)._**
  
- **_Resta (Sustracción)._**
  
- **_Multiplicación (Multiplicación)._**
  
- **_División (División)._**
